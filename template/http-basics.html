Well done, you have just made an HTTP request!

HTTP request are query sent from a client to a server. Most of the time, the client is your browser and the server is a website. You can demand various action from the server, the most common are GET, POST, PUT, DELETE. Most of the time, your browser uses GET to request specific page from the server.
