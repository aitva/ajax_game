Well done, you have just made an HTTP request!

HTTP request are query sent from a client to a server. Most of the time, the client is your browser and the server is a website. You can demand various action from the server, the most common are GET, POST, PUT, DELETE. The action must be follow by a path (/http-basics/, /wiki/Hypertext_Transfer_Protocol) and a protocol version (HTTP/1.1, HTTP2). Most of the time, your browser uses GET to request specific page from the server.

For more information go to Wikipedia.
