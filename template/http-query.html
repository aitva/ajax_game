Let's spice things up! Using just a verb and a path make HTTP looks quite boring. We will add additional information to our request so the server can introduce nuances in its response.

There are two ways to add information to a request: the fist one is the query string (adding information to the URL) and the second one the request's body (adding information inside the request).
